# Think Speccy, but for the CLI 

This is meant to be a tool to gather/detect hardware information
the same way [speccy by piriform](https://www.ccleaner.com/speccy/download) or [neofetch](https://github.com/dylanaraps/neofetch) does. 

This tool presents detailed software specs on your system using python. 

### Unicode Characters 
I added a few emojis. Theyare unicode characters which are part of the standard Python language and require no extra modules. Best to keep things simple. I may clean this up in the future.
